# Pluteus — Персональный трекер медиа-контента

Приложение для отслеживания фильмов, сериалов и книг на iOS с использованием SwiftUI и SwiftData.

## Возможности

### Основные функции
- **Добавление медиа**: фильмы, сериалы, книги
- **Автоматическое заполнение** через API:
  - TMDB (The Movie Database) — для фильмов и сериалов
  - Google Books API — для книг
- **Ручное добавление** с загрузкой обложки из галереи
- **Отслеживание прогресса**:
  - Для сериалов: количество просмотренных серий
  - Для книг: количество прочитанных страниц
- **Оценка** по 10-балльной шкале (звёзды)
- **Личные отзывы** и заметки
- **Фильтрация и сортировка** записей
- **Поиск** по своим записям
- **Свайп-действия** для быстрой смены статуса и удаления

### Технические особенности
- SwiftUI для пользовательского интерфейса
- SwiftData для локального хранения данных
- MVVM-архитектура
- Поддержка светлой и тёмной темы
- Локализация на русском языке
- Минимальная версия iOS 17

## Настройка API ключей

Перед запуском приложения необходимо получить API ключи:

### 1. TMDB API (для фильмов и сериалов)
1. Зарегистрируйтесь на https://www.themoviedb.org/
2. Перейдите в настройки аккаунта → API → Создать ключ
3. Выберите "Developer" тип ключа
4. Скопируйте полученный API key (v3 auth)

### 2. Google Books API (для книг)
Google Books API не требует ключа для базового использования, но имеет ограничения на количество запросов.

### Установка ключа

Откройте файл `Pluteus/Services/APIModels.swift` и замените:

```swift
static let tmdbAPIKey = "YOUR_TMDB_API_KEY"
```

на ваш реальный ключ:

```swift
static let tmdbAPIKey = "ваш_ключ_tmdb"
```

## Сборка и запуск

1. Откройте проект в Xcode 15 или новее:
   ```bash
   open Pluteus/Pluteus.xcodeproj
   ```

2. Выберите устройство или симулятор

3. Нажмите Run (⌘R)

## Использование

### Добавление записи

1. Нажмите кнопку "+" на главном экране
2. Выберите тип контента (фильм/сериал/книга)
3. Введите название в поле поиска
4. Нажмите на иконку лупы для поиска
5. Выберите найденный элемент из списка
6. При необходимости отредактируйте данные
7. Добавьте оценку и отзыв (опционально)
8. Нажмите "Сохранить"

### Просмотр и редактирование

- Нажмите на запись в списке для просмотра деталей
- Нажмите "Редактировать" для изменения данных
- Используйте свайп влево для быстрой смены статуса или удаления

### Фильтрация и поиск

- Используйте горизонтальные скроллы для фильтрации по типу и статусу
- Введите текст в поле поиска для фильтрации по названию
- Выберите опцию сортировки в выпадающем меню

## Архитектура

Приложение использует архитектуру MVVM:

- **Model**: Модели данных SwiftData (@Model)
- **View**: SwiftUI представления
- **ViewModel**: Классы @Observable для управления состоянием

## Хранение данных

Все данные хранятся локально с помощью SwiftData:
- Информация о медиа записях
- Изображения обложек (в виде Data)
- Прогресс просмотра/чтения
- Оценки и отзывы
